funciones module

class funciones.Modelo

Bases: object

Clase para realizar el CRUD sobre la base de datos

acerca()

Muestra una ventana emergente con información sobre el software y su autor.

alta(producto, stock, costo, venta, proveedor, tree)

Permite dar de alta a un registro en la base de datos «DB_PRODUCTOS» Se debe completar todos los campos para poder realizar el alta.

cambiar_fondo(aplicacion)

Permite al usuario cambiar el color de la ventana

consultar(pro, arbol)

Realiza una consulta a la base de datos «db_producto» Muestra los resultados en el TreeView. La consulta la realiza utilizando el producto.

eliminar(tree)

Elimina un registro de la base de datos utilizando el id seleccionado de una consulta que fue impresa en el treeview.

limpiar_tree(tree, entry_list)

LIMPIA EL TREEVIEWW DE TODA LA INFO QUE ESTE EN EL MOMENTO TAMBIEN LIMPIA LOS ENTRY DEL FORMULARIO

modificar(tree, pro, st, costo, venta, prov, entry_list)

Modifica un registro en la tabla «productos». Utiliza como parametros los valores del TreeView y las entradas correspondientes
